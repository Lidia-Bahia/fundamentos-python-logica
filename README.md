# Fundamentos de Programação com Python

Repositório com exercícios e implementações voltadas ao desenvolvimento de lógica de programação utilizando Python.

## Objetivo
Desenvolver raciocínio lógico e estruturar soluções computacionais para resolução de problemas, com foco em construção de base sólida para análise de dados.

## Conteúdos abordados
- Estruturas condicionais (if/else)
- Estruturas de repetição (for/while)
- Manipulação de dados básicos
- Funções e organização de código

## Programas incluídos

- **Gerenciador de Personagens**  
  Sistema simples para cadastro, consulta e remoção de personagens.

- **Calculadora de Juros**  
  Programa que calcula o valor final de uma compra parcelada aplicando diferentes taxas de juros de acordo com a quantidade de parcelas.

- **Sistema de Pedidos de Camisetas**  
  Simulação de um sistema de cobrança para pedidos de camisetas, considerando modelo, quantidade com desconto e frete.

- **Sistema de Pedidos de Lanchonete**  
  Programa que simula o atendimento de uma lanchonete, permitindo escolher pratos, tamanhos e calcular o valor total do pedido.

## Tecnologias utilizadas

- Python

## Observação

Os códigos foram originalmente desenvolvidos como atividades acadêmicas e posteriormente organizados para publicação neste repositório.
